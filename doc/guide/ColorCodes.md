# 色定数一覧

## 色定数一覧

これらの定数を様々な色指定の場面で使うことができます。これらの値は Windows の設定によって実際の色が変わる可能性があるものです。

```
`Scrollbar`clScrollBar           : スクロールバーの色
`Background`clBackground          : デスクトップの背景色
`ActiveCaption`clActiveCaption       : アクティブなウィンドウのタイトルバーの色
`InactiveCaption`clInactiveCaption     : アクティブでないウィンドウのタイトルバーの色
`Menu`clMenu                : メニューの背景色
`Window`clWindow              : ウィンドウの背景色
`WindowFrame`clWindowFrame         : ウィンドウの枠の色
`MenuText`clMenuText            : メニューの文字の色
`WindowText`clWindowText          : ウィンドウ内の文字の色
`CaptionText`clCaptionText         : アクティブなウィンドウのタイトルバーの文字の色
`ActiveBorder`clActiveBorder        : アクティブなウィンドウの境界線の色
`InactiveBorder`clInactiveBorder      : アクティブでないウィンドウの境界線の色
`AppWorkspace`clAppWorkSpace        : アプリケーションの作業領域の色
`Highlight`clHighlight           : 選択文字の背景色
`HighlightText`clHighlightText       : 選択文字の色
`ButtonFace`clBtnFace             : 3D オブジェクトの面の色
`ButtonShadow`clBtnShadow           : 3D オブジェクトの影の色
`GrayText`clGrayText            : 淡色表示されている文字の色
`ButtonText`clBtnText             : 3D オブジェクトのテキストの色
`InactiveCaption`clInactiveCaptionText : アクティブでないウィンドウのタイトルバーの文字の色
`ButtonHighlight`clBtnHighlight        : ボタンの強調表示の色
`ThreeDDarkShadow`cl3DDkShadow          : 3D オブジェクトの暗い方の影
`ThreeDHighlight`cl3DLight             : 3D オブジェクトの明るい面の縁の色
`InfoText`clInfoText            : ツールチップのテキスト色
`InfoBackground`clInfoBk              : ツールチップの背景色
```

> **Note:**
> 上記の色見本 ( 各色の左の `red` のような表示 ) は、ブラウザによっては正常に表示されません。
